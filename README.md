# SnapBudget 📸💰

**AI-Powered Personal Finance Management System**

SnapBudget is an intelligent budget tracking application that uses computer vision and deep learning to automatically extract transaction data from receipt photos.

**Built for**: Coding Camp 2026 DBS Foundation (CC26-PSU098)

---

## 🌟 Key Features

- **AI Receipt Scanning**: Automatic OCR and transaction categorization
- **Smart Budget Tracking**: Real-time spending analytics across 8 categories
- **Spending Predictions**: 7-day forecast using GRU models
- **Financial Insights**: Personalized budget recommendations

---

## 🛠️ Tech Stack

**Frontend**: React 19, Vite, Tailwind CSS, Recharts  
**Backend**: Node.js, Express, Prisma, PostgreSQL, Redis  
**AI/ML**: Python, FastAPI, PyTorch, TensorFlow, Hugging Face Transformers  
**DevOps**: Docker, Docker Compose, GitHub Actions
